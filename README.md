# Portfolio - J Manasa Krishna

A modern, interactive portfolio website built with React, TypeScript, and Tailwind CSS.
This project showcases my work as a Software Engineer and Frontend Developer, featuring 3D elements, smooth animations, and a responsive design.

## Tech Stack

- **React** - UI Library
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Three.js / React Three Fiber** - 3D Elements
- **Vite** - Build Tool

## Getting Started

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd portfolio
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```

## Customization

You can update the portfolio content by editing `src/data/portfolio.ts`. This single file contains all the data for the Hero, About, Experience, Projects, Skills, and Education sections.
